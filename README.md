<h1>AI Insurance Fraud Detection System 🚀</h1>

An intelligent microservices-based insurance fraud detection platform powered by AI, Kafka event streaming, JWT authentication, and real-time analytics dashboards.

This system detects suspicious insurance claims using AI-powered fraud analysis and visualizes fraud insights through a modern React dashboard.

✨ Features

✅ AI-powered fraud analysis using OpenRouter API <br>
✅ Event-driven microservices architecture with Kafka<br>
✅ JWT authentication & protected routes<br>
✅ Real-time fraud analytics dashboard<br>
✅ Dynamic charts and fraud visualization<br>
✅ Claim submission workflow<br>
✅ Automated fraud scoring & risk classification<br>
✅ Dockerized infrastructure<br>
✅ MySQL database integration<br>
✅ Modern responsive React frontend<br>
